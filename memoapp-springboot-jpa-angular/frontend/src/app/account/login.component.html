CommandMosaic Sample Application This application demonstrates the usage of CommandMosaic and its concepts: client code using the Command Dispatcher HTTP API and Java backend implemented as commands.
To Log in, use of the below credentials or register a new user.
  • Username: 'john', password: 'john1234': standard user
  • Username: 'fred', password: 'fred1234': standard user
  • Username: 'admin', password: 'admin1234': administrator user (permission to user management)
NOTE: The backend uses an in-memory database: all changes are reset and all memos stored are removed when the backend server is restarted. This behaviour is intentional for this sample application.
